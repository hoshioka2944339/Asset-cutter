# Asset Cutter

AI生成画像から複数の素材を切り抜き、背景透過、サイズ統一、PNG一括保存を行うブラウザ完結型ツールです。

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 画像の読み込み方

1. 右側パネルの「画像を読み込む」を押します。
2. PNG / JPG / WebP 画像を選びます。
3. 画像が左側の編集エリアに表示されます。

## 切り抜き範囲の作り方

1. 上部ツールバーの「範囲作成」を選びます。
2. 画像上をドラッグして矩形を作ります。
3. 「選択」を選ぶと、作成済みの範囲を移動・リサイズできます。範囲の内側をドラッグすると移動、四隅や辺のハンドルをドラッグするとリサイズできます。
4. 右側の「選択中の素材」で X / Y / W / H を数値入力すると、1px単位で調整できます。

## 背景透過の使い方

- 「透過なし」: 選択範囲をそのまま切り抜きます。
- 「単色背景透過」: スポイトで選んだ背景色に近い色を透明にします。
- 「外周連結背景透過」: 画像の外側とつながっている背景だけを透明にします。キャラ内部の近い色を消しにくい方式です。

「許容値」は、背景色にどれくらい近い色まで透明にするかを決めます。
「フチぼかし」は、切り抜き境界を少しなめらかにします。ドット絵では0推奨です。

## 自動検出の使い方

1. 背景透過設定で背景色と許容値を調整します。
2. 「最小面積px」「まとめ距離px」「検出余白px」を必要に応じて調整します。
3. 「自動検出」を押すと、背景ではない領域を素材候補として素材一覧に追加します。

小さな点が素材として出る場合は「最小面積px」を上げてください。近い装飾が別素材になりすぎる場合は「まとめ距離px」を上げてください。

## ドット絵モードの使い方

「ドット絵モード」をONにすると、プレビューや保存時の拡大縮小でドット絵がぼやけにくくなります。

- リサイズ方式は自動的に「ニアレストネイバー」になります。
- X / Y / W / H は整数に丸められます。
- 「元サイズのまま」「2倍」「3倍」「4倍」などの整数倍出力を選べます。

普通のイラストや写真では「なめらか」を使うと自然に見えやすくなります。

## 保存方法

1. 必要な素材範囲をすべて作ります。
2. 出力サイズ、余白、背景透過設定を調整します。
3. 「ZIPで保存」を押します。

ZIPには以下が含まれます。

- 各素材のPNG
- `preview_sheet.png`: プレビューシート
- `manifest.json`: 元画像内の座標や透過設定を記録した設定ファイル

「指定サイズに合わせる」を選んでいる場合、保存されるPNGは出力サイズで指定した正方形サイズになります。たとえば選択範囲が286x286pxでも、出力サイズが64x64ならPNGは64x64pxです。

## よくあるエラー

- 画像が読み込めない  
  PNG / JPG / WebP 形式か確認してください。

- 背景が消えすぎる  
  「許容値」を下げてください。キャラ内部の色まで消える場合は「外周連結背景透過」を試してください。

- 背景が残る  
  スポイトで背景色を取り直し、「許容値」を少し上げてください。

- ドット絵がぼやける  
  「ドット絵モード」をONにし、リサイズ方式を「ニアレストネイバー」にしてください。

- ZIP保存できない  
  選択範囲が1つ以上あるか確認してください。ブラウザのダウンロード許可も確認してください。
